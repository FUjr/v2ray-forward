# V2ray Forward Quickstart

## How to Use

1. Clone this repository

    ```bash
    git clone https://github.com/nofacer/v2ray-forward.git
    ```

2. Install v2ray

    **Online Install**

    ```bash
    wget https://install.direct/go.sh && \
    sudo bash go.sh
    ```

    **Offline Install**

    ```bash
    sudo bash go.sh -l v2ray-linux-<64/arm>.zip
    ```
